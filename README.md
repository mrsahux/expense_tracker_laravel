# Expense Tracker (Laravel)

This Expense Tracker is a web application built with the Laravel framework. It helps users manage and track their daily expenses efficiently. The application includes user authentication, expense categorization, and monthly filtering.

## Key Features

- **User Authentication**: Secure login and registration system using Laravel Breeze.  
- **Expense Management**: Users can add new expenses, edit existing ones, and delete expenses they no longer need.  
- **Monthly Filtering**: Easily filter expenses by month to see a summary for any specific month.  
- **Responsive Design**: Built with Bootstrap to ensure the app works well on different screen sizes.  
- **Secured Routes**: All expense-related routes are protected and accessible only to authenticated users.

## Technologies Used

- **Framework**: Laravel (PHP)  
- **Frontend**: Blade templates and Bootstrap for styling and responsive design.  
- **Database**: MySQL for storing user and expense data.  
- **Version Control**: Git and GitHub for source code management.
